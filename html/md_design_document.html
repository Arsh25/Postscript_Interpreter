Goal

Produce a C++ library that takes commands to draw certain shapes and outputs code in the PostScript programming language.

Testing Framework

We are using the Catch framework for our testing. This is an easy to use testing framework that only needs one header and one #include.

We are also utilizing Travis CI to automate the testing. When new code is pushed onto GitHub, Travis CI is called to build the testing code and report any errors.

Documentation

We are utilizing a documentation generation tool from source code called Doxygen. It generates an online documentation browser (HTML) which makes it easy to navigate the source code. It also makes it easy to see the inheritance and dependencies because of the automatic generation of graphs and diagrams.
